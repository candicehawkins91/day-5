Candice's Git Bash Cheat Sheet

Git Commands

Git init

Git init: initialize an existing directory as a git repository

Git clone

Git clone: retrieve an entire repsoritory from a hosted location via URL

Git add

Git add: adds a file as it looks now to your next commit

Git commit

Git commit: commits your staged content as a new commit smapshot

Git push

Git push: transmit local branch commits to the remote repository branch

Bash Commands

cd

cd: change directory

ls

ls: list directory content

touch

touch: creates an empty file

mkdir

mkdir: makes a directory

pwd

pwd: print current working directory
